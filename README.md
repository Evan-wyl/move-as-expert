# move-as-experts
该项目旨在梳理模仿学习、离线强化学习、扩散模型与强化学习、以及Transformer与强化学习相关知识，方便进一步学习。



本项目计划分为三个阶段，分别是

- 知识梳理阶段 $\leftarrow$ **正在进行**
- 算法复现阶段
- 项目优化阶段

知识梳理阶段主要关注知识框架的搭建；算法复现阶段主要关注经典算法的代码复现；项目优化阶段主要关注**知识完整性和准确性**、**排版整洁性**、以及**代码准确性**。

欢迎批评指正～

欢迎一起做项目～
